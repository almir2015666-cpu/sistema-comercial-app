Política de Privacidade

CMH Sistema — Clínica Medicina Humana
Última atualização: 19 de junho de 2026

Este aplicativo ("CMH Sistema") é uma ferramenta interna de gestão usada pela equipe da Clínica Medicina Humana para controle comercial e financeiro. Esta política explica quais dados são tratados ao usar o app.

Quais dados coletamos

  • Dados de login: e-mail e senha (autenticação via Firebase), usados apenas para identificar o colaborador que acessa o sistema.
  • Nome e função do colaborador cadastrados pelo administrador do sistema.
  • Dados operacionais da clínica: agendamentos, orçamentos, lançamentos financeiros (contas a pagar/receber) e cadastros de empresas parceiras, inseridos pelos próprios usuários durante o uso normal do sistema.

Como usamos esses dados

Os dados são usados exclusivamente para o funcionamento do sistema de gestão interna da clínica: controle de agenda, registro de atividades comerciais e controle financeiro. Não usamos os dados para publicidade, não vendemos nem compartilhamos dados com terceiros para fins de marketing.

Onde os dados são armazenados

Os dados são armazenados de forma segura no Firebase (Google Cloud), serviço de infraestrutura usado pelo sistema para autenticação e banco de dados.

Compartilhamento com terceiros

Não compartilhamos dados pessoais com terceiros, exceto os provedores de infraestrutura necessários para o funcionamento do app (Firebase/Google Cloud).
